# Incident-Analytics

SOC Incident Analytics Suite

Short description: Python toolkit for ingesting logs, running detections, enriching IOCs, and generating reports.

## Quick start
1. Create and activate a virtual environment
2. Install dependencies: `pip install -r requirements.txt`
3. Run demo: `python -m src.run_demo` or `./scripts/run_demo.sh`

## Project layout
- data/ sample logs
- src/ source code
- notebooks/ exploratory notebooks
- reports/ generated outputs
- config/ detection thresholds
- scripts/ convenience scripts
